# The Wheels in Motion

---

## Overview
The Wheels in Motion is a data-driven project focused on analyzing and visualizing vehicle-related data. The project processes raw data, performs data preparation, and generates insights through visualizations and dashboards.

---

## Features
- **Data Preparation**: Scripts for loading, cleaning, and merging raw data ([notebooks/data_preparation.ipynb](notebooks/data_preparation.ipynb)).
- **Visualization**: Dashboards and visualizations for insights ([src/visualization/dashboard.py](src/visualization/dashboard.py)).
- **Machine Learning**: Integration with Hugging Face for NLP tasks ([notebooks/huggingface.ipynb](notebooks/huggingface.ipynb)).
